# Richtlinien

# Plexiglas
- max. Größe 600x400 m^2
- transparent- oder Milchglas
- 3mm oder 5mm
# Hauptversorgung
- Netzteil
- MAX 12v
- PV-Anlage + Speicher
- Über PV-Anlage soll die Beleuchtung versorgt werden
# Arduino oder Raspberry PI

## Sensoren
- Temperatur
- Luftfeuchtigkeit

## Aktoren
- Lcd 16x2
- Beleuchtung
- Belüftung

## Zusatz Funktionen
- Raspberry Pi für Visualisierung auf einem lcd angeschlossen
